# Project EcustNet

### 简介 · Introduction

本项目唯一目的为服务学在华理的各位。

EcustNet为华理校园网（“ECUST”）的**自动连接程序**。

### 版本 · Version  

#### 1. Python(命令行)：run.py
使用方法：
1. 编辑run.py内学号信息
2. 然后日常需要连接校园网时，命令行中`python run.py`即可
